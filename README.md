## ゲームルール
エネルギーコアを持ってバッテリーにエネルギーを溜める。
エネルギーコアを投げて放電させ、相手を倒してエネルギーを奪う。
満タンになったバッテリーを大砲にセット、大砲を発射し相手を倒して勝利。
